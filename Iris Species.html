Iris Species Classification

Project Objective

This project focuses on analyzing the classic Iris flower dataset to explore relationships between petal and sepal dimensions and accurately classify flower species. Using Python, data visualization, and machine learning models, the goal is to identify key patterns and build a predictive model for species recognition.

K-Means Clustering and Hierarchical Clustering with PCA

Petal length and width were the most influential features for classifying species. The Random Forest model achieved the highest accuracy among tested algorithms, confirming strong separability between the three Iris classes.
